# How-To

## Windows

To run this project on Windows workstation do the following:

* Clone the project
```bash
cd /your/project/path
``` 
```bash
git clone https://github.com/j4ky/cypress-five.git
```
* Install Cypress
```bash
cd /your/project/path
```
```bash
npm install cypress --save-dev
```
* Run tests using Cypress UI
```bash
npx cypress open
```
